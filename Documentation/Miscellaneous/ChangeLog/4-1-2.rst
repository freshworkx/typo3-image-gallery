==========================
Version 4.1.2 - 2020/03/12
==========================

This release is a maintenance release. It contains mostly bug fixes.

Download
========

Download this version from the `TYPO3 extension repository <https://extensions.typo3.org/extension/bm_image_gallery/>`__ or from
`GitHub <https://github.com/bitmotion/typo3-image-gallery/releases/tag/v4.1.2>`__.

Added
=====

* Support for TYPO3 v10.3
* Fourth argument for :php:`getFileCollectionById()` method for sorting direction of files.

Changed
=======

* Author and company information
